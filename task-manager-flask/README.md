# Task Manager Flask

## Overview
Task Manager Flask is a simple web application built with Flask to help users manage their tasks in a To-Do List format. Users can add tasks, mark them as done, delete them, and search tasks in real-time. The project demonstrates basic web development using Python, Flask, HTML, CSS, and JavaScript.
